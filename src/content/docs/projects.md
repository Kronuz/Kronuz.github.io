## [Xapiand](https://github.com/Kronuz/Xapiand)

**Search and Storage Engine.** A fast, simple, and modern search and storage engine, written in C++.

🌐 [Website](https://Kronuz.github.io/Xapiand) · 📄 [Docs](https://Kronuz.github.io/Xapiand/docs) · [GitHub](https://github.com/Kronuz/Xapiand)

## [SublimeCodeIntel](https://github.com/SublimeCodeIntel/SublimeCodeIntel)

**Sublime Code Intelligence.** Full-featured code intelligence and smart autocomplete engine for Sublime Text.

🌐 [Website](http://www.sublimecodeintel.com) · 📄 [Docs](https://www.sublimecodeintel.com/docs) · [GitHub](https://github.com/SublimeCodeIntel/SublimeCodeIntel)

## [pyScss](https://github.com/Kronuz/pyScss)

**A Sass compiler for Python.** A compiler for the Sass language, a superset of CSS3 that adds programming capabilities and some other syntactic sugar.

📄 [Docs](http://pyscss.readthedocs.io) · [GitHub](https://github.com/Kronuz/pyScss)

## [Mech](https://github.com/mechboxes/mech)

**Supercharged Development Environments.** Easy command-line virtual machines. Like Vagrant, but for VMware, and free.

🌐 [Website](https://mechboxes.github.io/mech/) · 📄 [Docs](https://mechboxes.github.io/mech/docs/) · [GitHub](https://github.com/mechboxes/mech)

## [SublimeLinter](https://github.com/SublimeLinter/SublimeLinter)

**Sublime Text Framework for Linting Code.** A plugin for Sublime Text that provides a framework for linting code.

🌐 [Website](http://www.sublimelinter.com) · [GitHub](https://github.com/SublimeLinter/SublimeLinter)

A Django web interface for browsing multiple git repositories: commits, branches, file
trees, and diffs, in a clean UI. It was a Django take on the GitList idea, from the era
before hosted git browsers were a given on every box.

This one didn't go anywhere, and that's fine. Not every project earns a second life; some
are just scratch for an itch you had one week. I leave it up because the itch was real and
the code still reads cleanly.
